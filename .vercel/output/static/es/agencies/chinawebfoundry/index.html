Nuestra tesis

Una web para China no es una web traducida. Una copia traducida no es una copia localizada. Hospedar fuera de China es un suelo, no un techo. La mayoría de las marcas extranjeras lanzan en China traduciendo su web global. Seis meses después, el pipeline está vacío. Llevamos quince años reparando exactamente eso.
