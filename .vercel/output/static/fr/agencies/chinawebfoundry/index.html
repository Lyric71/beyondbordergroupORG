Notre conviction

Un site pour la Chine, ce n’est pas un site traduit. Une copie traduite n’est pas une copie localisée. Un hébergement hors de Chine est un plancher, pas un plafond. La plupart des marques étrangères se lancent en Chine en traduisant leur site global. Six mois plus tard, le pipeline est vide. C’est précisément ce que nous réparons depuis quinze ans.
